What is Javascript?

Javascript programs the behavior of web pages and is one of the most popular programming langaguges. It is used with HTML and CSS to create a functioning web page.

Where do you add Javascript?

There are a few different ways to place JavaScript. If you only want one file, you could place it in the script tag in html.

Or you could create another file and link it back to the HTML. For these examples, I will use another file.

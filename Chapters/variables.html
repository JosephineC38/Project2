Variables

To start, play a game of Flappy Bird . Hopefully you aren't that good with it. Now that you are done, what do you notice each time you pass a pipe?

Your score increase by one. That's because your score is a variable. A variable can store reuseable values. They can anything from the number of steps you walked or your favorite flavor of ice cream (obviously Rocky Road). There are several wyas you can initialize a variable.

  • const
  • let
  • var

const should be used if the value or type shouldn't be changed.

let should only be used if you can't use const

var is a local variable should only be used to support old browsers. A local varible can only exist within the funtion it was created in. That means the same name can be used in.

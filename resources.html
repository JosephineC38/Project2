Resources

This isn't the greatest guide and doesn't cover everything. So here are more online resources for you to discover.



w3schools

It's Javascript page gives detailed instructions on various methods and explains topics like Web APIs, JSON and Classes. It also has guides about HTML, CSS, Python and Java.


Mozilla Web Docs

Out of all the resources, this website is the most indepth about the various methods of Javascript.



Turner's Tutorials

It neatly organizes the complexity of programming a webpage into 9 levels. It also comes with samples and projects.


Runestone Academy

The Fundamentals of Web Programming gives the basics of HTML, CSS and Javascript. It also has various textbooks for topics like Python, Java, and Data Science.
